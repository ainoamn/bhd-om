# اختبار الموقع

## الخطوات للتحقق من أن الموقع يعمل:

1. تأكد أن السيرفر يعمل:
   ```bash
   npm run dev
   ```

2. افتح المتصفح على:
   - http://localhost:3000 (سيتم إعادة التوجيه تلقائياً)
   - http://localhost:3000/ar (العربية)
   - http://localhost:3000/en (الإنجليزية)

3. الصفحات المتاحة:
   - `/ar` - الصفحة الرئيسية
   - `/ar/projects` - المشاريع
   - `/ar/services` - الخدمات
   - `/ar/contact` - التواصل
   - `/ar/about` - عنا

## إذا استمرت المشكلة:

1. افتح Developer Tools (F12)
2. اذهب إلى Console
3. انسخ أي أخطاء تظهر
4. أرسلها لي
