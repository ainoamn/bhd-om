# دليل البدء السريع

## للتأكد من أن الموقع يعمل:

### 1. تأكد أن السيرفر يعمل:
```bash
npm run dev
```

### 2. افتح المتصفح على:
- **http://localhost:3000** (سيتم إعادة التوجيه تلقائياً إلى `/ar`)
- أو مباشرة: **http://localhost:3000/ar** (العربية)
- أو: **http://localhost:3000/en** (الإنجليزية)

### 3. إذا رأيت خطأ 404:

#### أ) تأكد أن السيرفر يعمل:
- ابحث في Terminal عن: `Local: http://localhost:3000` أو `Local: http://localhost:3002`
- افتح الرابط الذي يظهر في Terminal

#### ب) أوقف السيرفر وأعد تشغيله:
```bash
# اضغط Ctrl+C لإيقاف السيرفر
# ثم شغله مرة أخرى:
npm run dev
```

#### ج) نظف مجلد .next:
```bash
# أوقف السيرفر أولاً (Ctrl+C)
# ثم:
Remove-Item -Recurse -Force .next
npm run dev
```

### 4. الصفحات المتاحة:
- `/ar` - الصفحة الرئيسية
- `/ar/projects` - المشاريع  
- `/ar/services` - الخدمات
- `/ar/contact` - التواصل
- `/ar/about` - عنا
- `/ar/test` - صفحة اختبار بسيطة

### 5. إذا استمرت المشكلة:
1. افتح Developer Tools (اضغط F12)
2. اذهب إلى تبويب Console
3. انسخ أي أخطاء تظهر
4. أرسلها لي
