# دليل السياسات المحاسبية
# Accounting Policy Manual

## 1. الأساس المحاسبي (Accounting Basis)

- **أساس الاستحقاق (Accrual Basis)**: تُسجّل الإيرادات عند تحققها والمصروفات عند استحقاقها، بغض النظر عن تاريخ التحصيل أو الدفع.
- **معايير التقارير المالية الدولية (IFRS)**: يُطبق النظام مبادئ IFRS في القياس والعرض والإفصاح.

## 2. القيد المزدوج (Double Entry)

- كل معاملة مالية تُسجّل في جانبين: **مدين** و**دائن**.
- **المدين = الدائن** في كل قيد (قاعدة إلزامية).
- الدقة: منزلتان عشريتان (0.01).
- التسامح: ±0.01 للتقريب.

## 3. دليل الحسابات (Chart of Accounts)

| النوع | الرمز | الوصف |
|------|-------|-------|
| أصول | 1xxx | الصندوق، البنوك، العملاء، أصول أخرى |
| التزامات | 2xxx | الموردون، عربونات مستلمة، ضرائب |
| حقوق الملكية | 3xxx | رأس المال، أرباح محتجزة |
| إيرادات | 4xxx | إيرادات الإيجار، المبيعات، رسوم |
| مصروفات | 5xxx | تشغيل، صيانة، إدارية، رواتب |

## 4. القواعد الحاكمة (Governance Rules)

| القاعدة | التطبيق |
|---------|----------|
| لا عملية بدون قيد | كل معاملة تُرحّل تلقائياً عبر Posting Rules |
| لا قيد بدون مستند | القيود اليدوية مسموحة؛ قيود الفواتير/الإيصالات مرتبطة بمستند |
| لا تعديل بدون أثر تدقيقي | Audit Log يسجل كل تغيير |
| لا حذف نهائي | إلغاء أو عكس فقط |
| لا ترحيل لفترة مغلقة | Period Lock يمنع الترحيل |

## 5. سياسة الفترات المالية (Fiscal Periods)

- السنة المالية: 1 يناير إلى 31 ديسمبر (افتراضي).
- إغلاق الفترة: يمنع الترحيل على الفترة المغلقة.
- الإغلاق يُسجّل في Audit Log مع المستخدم والتاريخ.

## 6. سياسة الضرائب (Tax Policy)

- ضريبة القيمة المضافة (VAT): اختيارية (0%, 5%, 15%).
- الحساب المستخدم: 2200 (ضرائب مستحقة).
- تُحسب عند إنشاء الفاتورة/الإيصال وتُرحّل تلقائياً.

## 7. Posting Rules (قواعد الترحيل)

| نوع المستند | المدين | الدائن |
|-------------|--------|--------|
| إيصال/فاتورة | الصندوق/البنك | الإيرادات + الضريبة |
| دفعة | المصروفات | الصندوق/البنك |
| عربون | الصندوق/البنك | عربونات مستلمة |

## 8. العملة (Currency)

- العملة الافتراضية: ر.ع (OMR).
- المدعومة: ر.ع، ر.س، د.إ، $.

## 9. الإفصاح (Disclosure)

- البيانات جاهزة للإفصاح (Disclosure-Ready).
- التقارير: ميزان المراجعة، قائمة الدخل، الميزانية العمومية، التدفق النقدي.
