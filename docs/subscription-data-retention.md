# الاحتفاظ ببيانات المستخدم بعد انتهاء الاشتراك / تنزيل الباقة

## المتطلب
- بعد انتهاء اشتراك المستخدم (أو تطبيق تنزيل الباقة المجدول)، يحتفظ النظام **بنسخة من بيانات المستخدم لمدة شهر**.
- **بعد الشهر** يُنفّذ **أرشفة كاملة** لبيانات المستخدم (حسب السياسة المحددة: إخفاء أو نقل إلى أرشيف).

## التنفيذ الحالي
- تطبيق التنزيل التلقائي: يتم عند انتهاء `endAt` (اليوم التالي لانتهاء الاشتراك) تلقائياً عند استدعاء `GET /api/subscriptions/me` أو `GET /api/subscriptions` (لوحة الإدارة).
- الاحتفاظ والأرشفة بعد شهر: يُفترض تنفيذها لاحقاً عبر **Cron job** أو **scheduled task** يفحص الاشتراكات المنتهية/المُنزّلة ويطبّق سياسة النسخ ثم الأرشفة بعد 30 يوماً.

## مراجع في الكود
- `app/api/subscriptions/me/route.ts`: تعليق بجانب منطق تطبيق التنزيل المجدول.
- لوحة الإدارة `GET /api/subscriptions`: تطبيق تنزيلات منتهية لجميع المستخدمين عند فتح الصفحة.
