## الميزات المطلوبة:

### 1. بوابة المستأجر (تحديث شامل)
- ✅ عقودي (موجود)
- 💳 مدفوعاتي (موجود - جاهز لـ Thawani)
- 🔧 صيانة (موجد)
- 📄 مستنداتي (موجود)
- 🔔 **تنبيهات جديدة** — متأخرات الإيجار، الفواتير
- 📅 **تقويم** — مواعيد الدفع، تجديد العقود
- ✅ **مهام** — مهام مرتبطة بالمستأجر
- 💰 **المبالغ المستحقة** — إيجار متأخر، فواتير
- ⭐ **تقييمي** — تقييم الالتزام (إيجار/فواتير/صيانة)

### 2. بوابة المالك (تحديث شامل)
- ✅ عقاراتي (موجود)
- 👥 مستأجري (موجود)
- 💰 إيرادات (موجود)
- 🏢 **مباني** — كل مبنى مع وحداته
- 📊 **مصاريف** — مصاريف الصيانة والتشغيل
- 📈 **أداء المستأجرين** — تقييمات + متأخرات

### 3. نظام تقييم المستأجر الذكي
- ⭐ تقييم الإيجار: دفع بوقت / متأخر / متكرر
- ⭐ تقييم الفواتير: دفع بوقت / متأخر
- ⭐ تقييم الصيانة: طلب معقول / مفرط
- ⚠️ ملاحظات سلبية/إيجابية

### 4. إنشاء حسابات تلقائي
- من دفتر العناوين
- اسم مستخدم تلقائي: USR-{C/T/O}-{YYYY}-{NNNN}
- كلمة مرور عشوائية
- إرسال بالبريد/واتساب

### 5. توقيع عقود ذكي
- OTP عبر SMS/البريد
- توقيع إلكتروني بسحب
- QR code للتحقق
- سجل توقيعات
