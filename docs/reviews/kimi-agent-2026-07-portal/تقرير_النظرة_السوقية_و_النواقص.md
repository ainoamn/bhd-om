# القسم 2: النواقص الكبيرة ❌ (من ملفك)

## ❌❌❌ ناقص حرج (يؤثر على competitiveness)

### 1. بوابة دفع إلكتروني (Thawani/PayPal) ❌
**من ملفك:** "بوابة دفع ذكية ومرلة... بطاقات ائتمانية، تحويلات بنكية، محافظ إلكترونية"

**الوضع الحالي:** لا يوجد ANY نظام دفع

**التأثير:** 🔴 لا يمكن تحصيل إيجارات إلكترونياً — كل شيء يدوي

**الحل:** تكامل Thawani (الأقرب لعمان) أو Telr أو PayFort
```typescript
// lib/payment/thawani.ts
export async function createPayment(amount: number, description: string) {
  const response = await fetch('https://api.thawani.om/v1/checkout/session', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${process.env.THAWANI_SECRET_KEY}` },
    body: JSON.stringify({ amount: amount * 1000, // هلالات
      currency: 'OMR',
      description,
      callback_url: 'https://bhd-om.com/payment/callback',
    }),
  });
  return response.json();
}
```
