## 4. الحلول

### ✅ الحل 1: تقليل maxWaitMs من 2000ms إلى 200ms

**الملف:** `app/[locale]/admin/AdminLayoutInner.tsx`
**السطر:** 140

```typescript
// قبل:
const maxWaitMs = 2000;

// بعد:
const maxWaitMs = 200;  // ← 10x أسرع!
```

**التأثير:** من 2,000ms → 200ms (توفير 1.8 ثانية)

---

### ✅ الحل 2: استبدال getSession() بالـ session من useSession()

**الملف:** `app/[locale]/admin/AdminLayoutInner.tsx`
**السطور:** 137-154

```typescript
// قبل: getSession() يدوي + 2 ثواني تأخير
useEffect(() => {
  let alive = true;
  const maxWaitMs = 2000;
  const timeout = window.setTimeout(() => {
    if (alive) setSessionFetchSettled(true);
  }, maxWaitMs);
  void getSession().then((s) => {
    if (!alive) return;
    window.clearTimeout(timeout);
    setSessionFetchSettled(true);
    if (s?.user) setPeekSession(s);
  });
  return () => { alive = false; window.clearTimeout(timeout); };
}, []);

// بعد: استخدام useSession() فقط (يأتي من Provider)
// نحذف هذا الـ useEffect بالكامل!
// useSession() في السطر 88 يكفي!
```

**التأثير:** توفير 300-800ms + إزالة تأخير 2 ثواني

---
