# Legacy — نظام إدارة العقارات (monolith)

نسخة مصدرية من المشروع المحلي `C:\dev\عقود الايجار` — للمرجع أثناء الدمج في `bhd-om`.

## المحتويات

| المسار | الوصف |
|--------|--------|
| `bhd-real-estate.html` | الواجهة والمنطق (SPA monolith) |
| `scripts/bhd/` | وحدات JavaScript المقسّمة |
| `desktop/` | تطبيق Electron (يُستغنى عنه في الاستراتيجية الجديدة) |
| `apps/api/` | Cloud API (Express + PostgreSQL) |
| `server/` | خادم محلي للواجهة |
| `deploy/` | Docker Compose للإنتاج |
| `docs/` | وثائق المعمارية والنشر |

## ما لم يُرفع (محلي فقط)

- `node_modules/`
- `.env` وملفات الأسرار
- `backups/` و`data/`
- قواعد بيانات `*.db`

## الاستراتيجية

الموقع الرئيسي والإنتاج: جذر المستودع (Next.js + Prisma).  
هذا المجلد مرجع لنقل المنطق (عقود، حجوزات، محاسبة، طباعة) تدريجياً.
