## مرجع سريع — اللغة في الكود / Language in code

```javascript
// الدالة العالمية (~سطر 6622)
const t = (ar, en) => (appUiLanguage === 'en' ? en : ar);

// تبديل اللغة
toggleAppLanguage()  →  localStorage: bhd_ui_language = 'ar' | 'en'
applyAppLanguage()   →  body.lang-ar | body.lang-en + localizeBilingualUi()
```

**عند إضافة واجهة جديدة:**
- نصوص ديناميكية (JS): `t('نص عربي', 'English text')`
- عناصر HTML ثابتة: `data-ar="..." data-en="..."`
- بعد بناء DOM ديناميكي: استدعِ `localizeBilingualUi()` إن لزم
- **ممنوع:** نص عربي فقط في `innerHTML` بدون `t()`
