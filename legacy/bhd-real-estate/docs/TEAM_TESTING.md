# اختبار الفريق — 15 مستخدماً على الشبكة المحلية

## 1) على جهاز الخادم (مرة واحدة)

```powershell
تشغيل-السحابة.cmd
node tools\enable-lan-access.js
تشغيل-الخادم.cmd
```

شارك الرابط الذي يظهر، مثال: `http://192.168.1.50:3789/`

> البروكسي السحابي يعمل من الخادم — الموظفون **لا يحتاجون** منفذ 3790.

## 2) على كل جهاز موظف

| الطريقة | ماذا يفعلون |
|---------|-------------|
| **متصفح (مُفضّل)** | افتح الرابط أعلاه — نفس الواجهة |
| **Desktop** | `cloudApiUrl` = `http://IP-الخادم:3790` في `bhd-app-config.json` |

## 3) قائمة اختبار (أسبوع 3)

- [ ] 3 مستخدمين يفتحون لوحة التحكم معاً
- [ ] كل واحد يعدّل وحدة مختلفة — يظهر التعديل بعد تحديث الصفحة
- [ ] مستخدمان على نفس العقد — آخر حفظ يفوز
- [ ] قيد محاسبي + طباعة A4
- [ ] إيقاف Docker 5 دقائق — Desktop يعمل — إعادة التشغيل — المزامنة تعود

## 4) فحص تقني (مدير النظام)

```powershell
cd apps\api
npm run load:test -- --users 15 --duration 30
npm run verify:full
```

## 5) مشاكل شائعة

| المشكلة | الحل |
|---------|------|
| لا يفتح من جهاز آخر | `HOST=0.0.0.0` + جدار الحماية منفذ 3789 |
| تسجيل دخول فاشل | نفس حسابات النظام الحالية |
| بيانات قديمة | تحديث الصفحة (F5) — المزامنة عند العودة للتبويب |
